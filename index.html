FAQ

Do I need to license any additional software like ArcGIS to run Vetro?
No, Vetro is an entirely secure, cloud-based application that is built using open source tools and does not require any 3rd party licensing.
Where is my data being stored?
All data is stored securely on an Amazon Elastic Computing (EC2) instance.
How will I be charged for using Vetro?
You will pay for Vetro through a monthly subscription. The price depends on the size of your network - not the number of users.
What does Vetro mean?
Vetro is the Italian word for glass, the material that makes up a fiber strand.
